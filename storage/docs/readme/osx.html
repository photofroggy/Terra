Welcome!

Welcome to the Mac OS X setup page! This page details how to get Terra working properly on an Apple computer, using the OS X Operating System.

Instructions

I can't actually write setup instructions for OS X, as I do not own a Macintosh, and have thus far been unsuccessful in getting help with writing instructions for OS X. All I can tell you is that the Python version that comes with OS X is outdated, and you are required to install Python 3 before running Terra.

There are disk image files available for Python on OS X on this website. A disk image is a like an installer for OS X which uses a point and click interface. You can download a disk image of Python 3 from the link below: Python 3.1.2 OS X Disk Image

Once you have Python 3 installed on your system, setting up Terra should be quite similar to setting up Terra on Linux. Because of this, this page may be of some use to you.
