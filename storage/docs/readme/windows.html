Welcome!

Welcome to the Windows setup page! This page details how to get Terra working properly on a computer using a Windows Operating System (XP, Vista, Windows 7).

Instructions

I have tried to make the setup instructions as easy to follow as possible. If you get stuck, feel free to ask someone for help, but keep in mind that you probably shouldn't be using the bot if you can't get it working following the provided instructions.

Install Python

Before you can get Terra up and running, you must first install Python. Luckily, this is an incredibly easy thing to do. Just follow the instructions below.

1 — Download An Installer

The first thing you need to do, is download Python! You need Python 3.0 or higher to run this bot! Python 3.1.1 for Windows (32Bit) is available in the link below: Python 3.1.1 Windows Installer

If you need a different installer, head over to this page and grab an installer for the latest release of Python 3.

2 — Run The Installer

Now that you have downloaded Python, you need to install it! To install it, simply double click the installer you downloaded, and follow any instructions given to you.

Setting Up Terra!

If you managed to install Python, you are very close to having your very own copy of Terra up and running! Getting Terra set up properly shouldn't be too difficult, so just follow the instructions below.

3 — Download Terra

First thing you need to do is download Terra. If you are reading this file, it is likely that you have already done so, but just incase you haven't, you can get the latest version of Terra at the website linked below: Terra's Botdom Wiki Page

4 — Extract Files

Once you have downloaded Terra, you will need to extract the contents of the downloaded file into a folder of your choice.

To do this, find the file you just downloaded, right click and select "Extract files..." (or "Extract all" depending on which version of Windows you are using).

Make sure your extract the bot to a folder that is easy to find. Something like "C:\Terra\" will do nicely.

5 — Run Terra

When you have completed step 3, go to the folder you extracted Terra to, and run Terra by double clicking on the file called Launch.py. A black box like the one below should appear on the screen. This black box is Terra itself!

The first time you start Terra, you will be required to configure it. To do this, follow the instructions given by the bot itself, in the black window. You are only required to do this the first time your run Terra.

Congratulations!

If everything went correctly, you now have a copy of Terra running on your computer!

As a final note, double clicking on any of the other files ending in .py inside your bot's folder will allow you to do different things. For examle, running Menu.py will bring up a main menu, allowing you to pick from a set of options.

Thanks for reading, and have fun!
