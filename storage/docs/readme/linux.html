Welcome!

Welcome to the Linux setup page! This page details how to get Terra working properly on a computer using a distribution of Linux. This particular guide was written based on the Ubuntu distribution of Linux, but the instructions may be similar on other distributions.

Instructions

I have tried to make the setup instructions as easy to follow as possible. If you get stuck, feel free to ask someone for help, but keep in mind that you probably shouldn't be using the bot if you can't get it working following the provided instructions.

1 — Install Python

Python 3.0 or higher is required to run Terra on Linux. Please note, you should not be required to set up Python 3 as your default Python installation. To install Python 3, open up the terminal and type the following command: sudo apt-get install python3

This may differ on different distributions of Linux!

2 — Download Terra

Now, you must download Terra. If you are reading this file, it is likely that you have already done so, but just incase you haven't, you can get the latest version of Terra at the website linked below: Terra's Botdom Wiki Page

3 — Extract Files

Before you can actually get Terra working, you need to extract the files from the zip file you just downloaded. To do this, use the following command in the terminal: unzip <downloaded file> -d <destination>

Where <downloaded file> is the full pathname of the file you just downloaded, and <destination> is the directory you want to extract the files to.

For example, if you downloaded the file to ~/Downloads and wanted to extract the files to ~/Desktop, then the command may look something like this: unzip ~/Downloads/Terra_1.zip -d ~/Desktop

4 — Changing Directory

Before you can do anything else, it is a good idea to change your working directory to the bot's home folder. To do this, you can type the following command into the terminal: cd <bot folder>

Where <bot folder> is the directory you extracted your files to in step 3, plus the sub-folder Terra. For example, if you extracted your files to ~/Desktop, then the command may look something like this: cd ~/Desktop/Terra

5 — Run Terra

After you have changed directories to the bot's folder, you can start it up! To do this, you will need to type the following command into your terminal: python3 Launch.py

If everything went according to plan, you should see something similar to the image below.

To complete the setup process, you will have to give Terra the information it asks for. After that, you can run Terra again in the same way, and it won't ask you for anything!

Congratulations!

If everything went correctly, you now have a copy of Terra running on your computer!

As a final note, running other files ending in .py the same way you did in step 5 will allow you to do different things. For examle, running Menu.py will bring up a main menu, allowing you to pick from a set of options.

Thanks for reading, and have fun!
